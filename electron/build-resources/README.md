# Build Resources

Place your app icons here for Electron builds:

- `icon.icns` — macOS icon (1024×1024)
- `icon.ico` — Windows icon (256×256)
- `icon.png` — Linux icon (512×512)

You can generate all formats from a single 1024×1024 PNG using tools like:
- [electron-icon-builder](https://www.npmjs.com/package/electron-icon-builder)
- [makeicns](https://formulae.brew.sh/formula/makeicns) (macOS)
